Небольшая инструкция для тех, кто хочет добавить свои картинки.
- Для правильной работы нужны два файла: back.png и mask.png одинакового размера.
back.png - сама картинка которая будет видна на заднем фоне
mask.png - "теневая картинка", которая будет исползована программой для симуляции снега.

как создать mask.png?
- все места, где снег должен проходить свободно, следует закрасить черным цветом
- Все препятствия для снега следует закрасить люыми цветами, отличными от черного.
